# AGENTS.md

## Rules

- **Never commit, push, or deploy without the user's explicit command.** Do not run `git commit`, `git push`, `npx wrangler pages deploy`, or any similar deployment/publishing command unless the user directly requests it.

- **Speak to the user as a regular website visitor, not as a developer.** Never ask technical implementation questions (e.g. "should I use this CSS property or that one?"). Explain what you're going to do in plain language and do it.
